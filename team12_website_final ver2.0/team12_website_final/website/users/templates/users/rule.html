網路交友安全守則

一、不可任意在網路上留下真實姓名、電話、單位住址、身分及服務單位等資料。
二、和網友相約見面時應另外找朋友作陪，避免單獨與網友見面。
三、和網友見面地點應選擇明亮、人多、交通方便且自己熟悉的公共場所。
四、若網友藉故要更改見面地點，例如改至對方家中或偏遠僻之地，則千萬不可答應。
五、不喝來路不明的飲料，離座回來後亦不再吃（喝）之前未吃（喝）完的食物（飲料）。
六、小心網路交友詐騙，在網路上的人不一定會說實話，常為虛擬身分或犯罪。
七、注意別讓自己成為受害。
八、不可隨便將信用卡帳號登錄在網路上，即使是父母的信用卡號亦同，自己的網路帳號密碼應妥善保管，不可給予他 人（括自己的朋友也不可以）。
九、不輕易將個人照片寄給他人。
十、接到任何令自己感到不舒服的訊息或信件時不需理會它，必要時請幹部或父母協助處理。
十一、避免和陌生網友有金錢上的往來。
十二、人格如網格，作網路的紳士淑女不要用粗俗不雅與挑逗的語言交談並拒絕和使用這類語言的人交往。
十三、天下沒有白吃的午餐，如果有人提供好得難以置信的禮物或差事時，就是要特別小心的時刻。
十四、不做網路大野狼，網有網法，做你自己，不要假冒不實的身分，網路與真實世界一樣，利用電子信箱誹謗、恐嚇 他人，是犯罪的行為。
十五、做e世代好公民，發現犯罪或可疑行為，請打110。
